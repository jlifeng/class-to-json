# Java Generate Json Schema Changelog


## [0.0.1] - 2025-03-22

### Added

- 将选中的多个 Java 类转换为 JSON Schema 的 JSON 文件
- 输出文件路径在项目根目录的 .json文件中，一个类对应一个文件

## [0.0.2] - 2025-03-25

### Added

- 支持更多类型，包括枚举类型，数组类型，集合类型，Map类型等
